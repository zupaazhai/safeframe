Explanation of Boot Loader

The Bootloader is a sample that shows a single script include configuring SafeFrame and delivering the ad. This page has two bootloader tags in place. Each tag specifies the ad script to be delivered and placement of the ad via fragment parameters on the script src attribute. 

The code

<script src="sf_bootloader.js#w=120&h=600&target=skyscraper_ad&ad=http://l.yimg.com/pq/js/adx_qa,cgong-teset/120_600_exp,C=demo,P=publisher1/ad2.js?q=1299740219"></script>

The bootloader script detects the presence or absence of the SafeFrame framework. If missing it dynamically loads the framework files. Once loaded the bootloader parses parameters passed to the script on the src URL fragment. These parameters specify location, size, and ad script URL.

ParameterDescription
wWidth of the ad
hHeight of the ad
adURL of the script that will deliver the ad content. In the bootstrap sample, this should be the last parameter.
